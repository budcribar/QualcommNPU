Windows ARM64X Support¶

Introduction

The Qualcomm® Neural Processing SDK supports ARM64X binaries for SC8380XP on Windows.

The following sections demonstrate this support in Qualcomm® Neural Processing SDK:

  1. ARM64X Library Supported Scenario

  2. ARM64X Usage

ARM64X Library Supported Scenarios

These sections demonstrate two supported scenarios.

  • ARM64/ARM64EC/ARM64X app – These 3 kinds of apps can be linked with ARM64X SNPE.lib and executed with ARM64X SNPE.dll.

  • x86_64 app – x86_64 app can be linked with x86_64 SNPE.lib and executed with ARM64X SNPE.dll.

ARM64X Usage

Tools, APIs, tutorials, and model conversion are the same as for ARM64 platforms.
