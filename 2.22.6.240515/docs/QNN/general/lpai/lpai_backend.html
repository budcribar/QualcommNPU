LPAI¶

This section provides information specific to QNN LPAI backend.

API Specializations¶

This section contains information related to API specialization for the LPAI backend. All QNN LPAI backend specialization is available under <QNN_SDK_ROOT>/include/QNN/LPAI/ directory.

The current version of the QNN LPAI backend API is:

QNN_LPAI_API_VERSION_MAJOR 2

QNN LPAI API Version values.

QNN_LPAI_API_VERSION_MINOR 8
QNN_LPAI_API_VERSION_PATCH 0

QNN LPAI Supported Operations¶

QNN LPAI supports running quantized 8-bit and quantized 16-bit networks on supported Qualcomm chipsets. List of operations supported by QNN LPAI Quant runtime can be seen under Backend Support LPAI column in Supported Operations.
