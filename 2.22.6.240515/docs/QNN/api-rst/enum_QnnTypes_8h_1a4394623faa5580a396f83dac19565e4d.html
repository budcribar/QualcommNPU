enum Qnn_Priority_t¶

An enum to specify a priority.

Values:

enumerator QNN_PRIORITY_LOW = 0¶

QNN_PRIORITY_LOW is always available for use.

enumerator QNN_PRIORITY_NORMAL = 100¶

QNN_PRIORITY_NORMAL is always available for use.

enumerator QNN_PRIORITY_DEFAULT = QNN_PRIORITY_NORMAL¶
enumerator QNN_PRIORITY_NORMAL_HIGH = 150¶

QNN_PRIORITY_NORMAL_HIGH usage may be restricted and would silently be treated as QNN_PRIORITY_NORMAL

enumerator QNN_PRIORITY_HIGH = 200¶

QNN_PRIORITY_HIGH usage may be restricted and would silently be treated as QNN_PRIORITY_NORMAL

enumerator QNN_PRIORITY_UNDEFINED = 0x7FFFFFFF¶
